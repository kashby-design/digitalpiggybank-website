## Production workflow per post

1. **Week before publish:** Draft 80% in Markdown. Add frontmatter (title, description, image, imageAlt, tags). Use placeholder image initially.
2. **Three days before:** Custom hero image commissioned (or pulled from a stock source with rights).
3. **One day before:** Final edit pass. Run a Lighthouse + Playwright suite (`pnpm build && pnpm test`).
4. **Publish day:** Push to main. Vercel deploys. Verify in production. Cross-post to social (FB, IG, TikTok, X).

## Distribution per post

- Push to all 4 social channels with a 2-sentence hook + image
- Pin to top of the blog index for 72 hours
- Mention in any active email newsletter (when set up)
- For high-effort posts (>1,200 words), do 1–2 manual outreach emails to relevant parenting blogs offering the post as a resource
